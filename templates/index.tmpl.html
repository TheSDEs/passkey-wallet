How it works

  • Piggybank uses Turnkey to create private keys.
  • These private keys are controlled by an API key generated on your device and encrypted with your Personal Passphrase.
  • Piggybank keeps your encrypted API key. Only you can decrypt it!
  • In case you lose access or forget your passphrase, you can generate a new API key, encrypted to a new Personal Passphrase. Piggybank will disable your old API key and replace it with the new one.
